SearchClip

This utility lets you search from any Windows application which has selectable text. When SearchClip is running, it monitors the keyboard and if ctrl-c is pressed twice within the mouse double click time then it launches a browser of your choice to search for the selected text (using a search engine of your choice).

If you're using GitHub for Windows, simply sync this repository and you'll see the new branch. Alternatively, just get the 1.0 release which is already compiled.
